Composites

composite SimpleExtractDomain(output OutputStream; input DomainNamesStream)

Composite to fill TLD of ExtractDomain operator with files scanned in tldsDir

Parameters

  • domainNameAttr: specifies the input attribute containing the FQDN that the extraction will be performed on.
  • outputAttr: specifies the output attribute to write the extracted domain+TLD data to
  • tldsDir: directory containing TLDS files
  • tldsFilenamePattern: file name pattern
  • blankOnInvalidTLD: default true

Input Ports

  • DomainNamesStream: tuples containing FQDNs

Output Ports

  • OutputStream: tuples with domain+TLD extracted from the FQDN field
