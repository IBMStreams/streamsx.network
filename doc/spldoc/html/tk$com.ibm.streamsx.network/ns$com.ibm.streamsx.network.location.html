Namespace com.ibm.streamsx.network.location

Toolkits > com.ibm.streamsx.network 3.0.0 > com.ibm.streamsx.network.location

Operators

  • IPAddressLocation: IPAddressLocation is an operator for the IBM InfoSphere Streams product that finds the geographical location of IP addresses received in input tuples, based on the subnets they are part of, and emits output tuples containing the country, state or province, city, latitude, and longitude of the subnets.
